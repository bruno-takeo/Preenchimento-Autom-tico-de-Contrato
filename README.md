Preenchimento Automático de Contratos
Este projeto é um aplicativo/software que facilita o preenchimento automático de contratos com base em inputs fornecidos pelo usuário. Ao fornecer as informações necessárias, 
o usuário pode gerar um novo arquivo de contrato personalizado, nomeado de acordo com sua preferência.

Funcionalidades
Interface de Usuário Amigável: O usuário pode inserir suas informações de forma intuitiva e fácil.
Preenchimento Automático: Integração das informações fornecidas pelo usuário diretamente no template do contrato.
Geração de Arquivo: Criação de um novo arquivo de contrato personalizado, que pode ser baixado pelo usuário.
Nomeação Personalizada: O usuário pode nomear o arquivo gerado conforme sua preferência.
